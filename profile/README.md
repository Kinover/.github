**Kinover**는 가족만을 위한 비공개 SNS입니다.  
일상을 공유하고, 감정을 표현하며, 추억을 기록하는 따뜻한 모바일 공간을 만들고 있어요.

## ✨ 주요 기능
- 실시간 채팅 & 접속 상태 확인
- 감정 표현 기능 (기분 공유)
- 가족 일정 공유
- 이미지 기반 추억 기록 (갤러리)
- AI 챗봇과의 대화

## 🛠️ 기술 스택
- React Native (Frontend)
- Spring Boot, Redis, MariaDB (Backend)
- AWS EC2 / S3 / RDS 기반 인프라
- GitHub Actions + Nginx 배포 자동화
